== Description ==
Used as an extension to the Snax plugin. This prevents the modifications to the 
original plugin, and provides a better method for safer updates.

== Installation ==

This section describes how to install the plugin and get it working. 

1. Upload `snax-mod.php` to the `/wp-content/plugins/snax-mod/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
